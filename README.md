# KillThaMail
Tool for finding and removing email objects in an exchange database. Requires Exchange 2013/2016 and "Microsoft.Exchange.Management.PowerShell.SnapIn".
Built using Sapien PowerShell Studio

There is a compiled .exe version of this available in the 'bin' folder. Otherwise, KillThaMail.Package.ps1 contains all of the source code for this project.
